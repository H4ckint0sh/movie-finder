# movie-finder

## Requirements: 

* Node (https://nodejs.org/en/)
* Expo (https://docs.expo.io/workflow/expo-cli/)
* Xcode - To run the simulator on iOS (app is available on the app store)
* Visual Code - We preffer Visual Code (https://code.visualstudio.com/)

## How to setup / run application:

* git clone ;)
* Always run ```npm i```before starting the project to get all packages installed
* ```npm start``` or ```expo start``` to start the application
* Have fun and enjoy your stay!
